Iniciantes com dúvidas

Se você já pesquisou sobre tecnologia, com certeza já ouviu falar em Front-End, Back-End e Full Stack, mas você sabe qual é a diferença entre Front-End, Back-End e Full Stack? Muitos programadores e programadoras iniciantes ficam com muitas dúvidas sobre quais são as diferenças entre eles. Então pega a pipoca e vamos dar uma conferida! 💻



O que é DEV? Ou Desenvolvedor/Programador?


DEV (ou desenvolvedor/programador) é o profissional que desenvolve sistemas, sites ou aplicativos para celulares. São pessoas que escrevem códigos em linguagens de programação, seja ela Python, JavaScript e entre outras.


Desenvolvedor Front-End


O desenvolvedor Front-End é responsável pela parte visual ou gráfica e a parte de interação do site, é toda a parte que o usuário interage com a aplicação. Sabe aquela interface super legal que vemos quando entramos, por exemplo, em alguma rede social? Pois é, são os profissionais da área front-end que fazem toda a interação e design da página.


O que um desenvolvedor Front-End precisa saber?


  • HTML
  • CSS
  • JavaScript
  • PHP

Desenvolvedor Back-End


O desenvolvedor back-end é o encarregado de manter o site ou software funcionando. Cuida do funcionamento de dados no servidor. É a parte que o usuário utiliza, mas não consegue ver ou perceber o funcionamento como vê no Front-End. O Back-End cuida de processamento de dados, regra de negócios, validações. E assim, tendo uma concatenação com o banco de dados e o Front-End


O que um Desenvolvedor Back-End precisa saber?


  • Lógica de Programação
  • Alguma linguagem de programação (Python, Java, C, C++)
  • Servidor
  • Banco de dados
  • Segurança
  • Escalabilidade

O Desenvolvedor FullStack

O desenvolvedor Full Stack atua no desennvolvimento da parte visual e funcional do site ou aplicativo. É responsável por trabalhar com Front-End, Back-End e Banco de Dados, ele deve ter conhecimentos em linguagens de Front-End (HTML5, CSS3, JavaScript, Angular), Back-End (Python, PHP) também Banco de Dados
